# streamlit_aircraft_crashes_dashboard2
Interactive Streamlit dashboard analyzing global aircraft crash data (2024) with insights, visualizations, and KPIs.
This repository contains a comprehensive interactive dashboard for analyzing global aircraft crash data using Python and Streamlit. The dashboard provides:

Key Insights and Metrics: Total crashes, fatalities, average fatalities per crash, and the most dangerous countries.

Interactive Visualizations:

Crashes per year (line chart)

Top 10 countries by crashes (horizontal bar chart)

Top 10 aircraft manufacturers (horizontal bar chart)

Fatalities vs. people aboard (scatter plot)

Survival rate distribution (histogram)

Crashes by month and per decade (bar charts)

Filters: Users can filter data by country, manufacturer, month, and year range.

Top 10 Deadliest Crashes Table: View detailed information about the most fatal incidents.

Technologies Used:

Python 3.x

Streamlit for interactive web app

Pandas for data manipulation

Plotly for interactive charts

Purpose:
This project is designed to provide aviation analysts, researchers, and enthusiasts with an easy-to-use interactive tool to explore trends in aircraft crashes and derive actionable insights.
